# **App Name**: TrackFlow

## Core Features:

- Add Expense: Allows the user to add a new expense with a title, amount, category, and date.
- Total Expense Display: Show the sum of all expenses at the top of the application.
- Expense Listing: List all expenses in a visually appealing manner, using animated cards to represent each expense item.
- Delete Expense: Enable users to delete individual expense entries from the list.
- Data Persistence: Use local storage for data persistence, ensuring data is saved and available offline.

## Style Guidelines:

- Primary color: Deep sky blue (#007BFF) for a modern and trustworthy feel.
- Background color: Light gray (#F8F9FA) to ensure good contrast and readability.
- Accent color: Cyan (#00FFFF) for interactive elements and highlights.
- Font: 'Inter' sans-serif for a clean, modern, fintech-style look across all text elements.
- Use minimalistic and consistent icons to represent expense categories.
- Implement a clean, modern layout with clear visual hierarchy for easy navigation.
- Apply slide-in and fade animations for expense cards, number transitions for total expenses, and ripple effects on buttons.
